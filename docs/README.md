# MemeRoo - Documentos Legales

Este directorio contiene los documentos legales de MemeRoo para GitHub Pages.

## Archivos

- `index.html` - Página principal con enlaces a los documentos
- `privacy-policy.html` - Política de Privacidad
- `terms-of-service.html` - Términos de Servicio

## URLs para Google Play Console

Una vez activado GitHub Pages, las URLs serán:

- Política de Privacidad: `https://yohan201609-oss.github.io/MemeRoo/privacy-policy.html`
- Términos de Servicio: `https://yohan201609-oss.github.io/MemeRoo/terms-of-service.html`

## Cómo activar GitHub Pages

1. Ve a tu repositorio en GitHub: https://github.com/yohan201609-oss/MemeRoo
2. Ve a **Settings** → **Pages**
3. En **Source**, selecciona la rama `main` y la carpeta `/docs`
4. Haz clic en **Save**
5. Espera unos minutos y las páginas estarán disponibles
